# 23. 文件存储共享服务配置与实战
本章


存储：
  DAS：Direct Attached Storage
    接口类型：“block”
    设备：SATA，SAS；IDE，SCSI；USB；
  NAS：Network Attached Storage
    接口类型："file"
    协议：CIFS(samba), NFS(Network File System)
    RPC: Remote Procedure Call
  SAN：Storage Area Network
    接口类型："block"
    协议：iSCSI(IP-SAN), FCSAN, FCoE, ...
  应用层：
    ftp, ...
