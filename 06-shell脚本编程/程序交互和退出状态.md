

## 2. bash 编程之用户交互
read [-ers] [变量名 ...]
- 作用:从标准输入读取一行并将其分为不同的域
- [-a 数组]
- [-d 分隔符]
- [-i 缓冲区文字]
- [-n 读取字符数]
- [-N 读取字符数]
- [-p 提示符]
- [-t 超时]
- [-u 文件描述符]
- eg:
    - read -p "enter a disk filename"  -t  5  name
    - [  -z  "$name"  ]   &&  name="value"

## 3. bash 调试
- bash  -n  script.sh  -- 检查脚本语法错误
- bash  -x  script.sh  --  单步执行，显示代码执行的详细过程


## 4. 脚本的状态返回值
默认是脚本中执行的最后一条件命令的状态返回值；
自定义状态退出状态码：
exit  [n]：n为自己指定的状态码；
注意：shell进程遇到exit时，即会终止，因此，整个脚本执行即为结束；
